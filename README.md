### 🧠 [Smart Datacenter: High Availability & AI-Driven Self-Healing](https://github.com/UKNJHB/PFM105.Smart-Datacenter-Self-Healing)
*A cutting-edge Graduation Project (PFM) combining Advanced Resiliency and Cognitive Automation.*
* **🎯 Live Demo:** Check out the interactive [Project Dashboard Webpage](https://uknjhb.github.io/PFM105.Smart-Datacenter-Self-Healing/) 👈 
* **Core Infrastructure:** Virtualized environment deployed on **VMware ESXi**, leveraging isolated Ubuntu Servers under managed NAT and Static IPs.
* **High Availability:** Sub-3-second network failover mechanism using **Keepalived (VRRP)**.
* **Cognitive Self-Healing:** Automated **n8n event pipeline** that extracts logs, integrates local **Llama 3 (Ollama)** for semantic failure diagnostics, and auto-recovers crashed services (Nginx/MySQL) in < 20 seconds.
